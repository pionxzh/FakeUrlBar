偽造網址列

     在手機瀏覽器中，許多瀏覽器都會在下滑一段距離後將網址列隱藏以達到更好的觀看體驗。

     自然而然，這個部分也成了釣魚網頁的攻擊重點，Jim fisher 正好發現了一種偽造“網址列”的方法，這種新型態的攻擊被稱之為Inception bar！

     這個repo實作了原文DEMO 中所詳細解釋的攻擊手法，並簡單封裝幾個API方便操作與展示。
